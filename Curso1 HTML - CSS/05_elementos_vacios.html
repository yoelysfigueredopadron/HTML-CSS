Bienvenido al curso Information Technology (IT) Essentials. El objetivo de este curso es presentar hardware y software de computación, así como sistemas operativos, conceptos de la red, dispositivos móviles, seguridad de TI y solución de problemas.

Este material del curso en línea lo ayudará a desarrollar las aptitudes necesarias para trabajar como técnico en el campo de TI.

Las habilidades específicas desarrolladas en cada capítulo se describen al comienzo de cada uno de ellos.
